MadCommerce is a simple eCommerce web application built using Vanilla JavaScript, focusing on core eCommerce functionality without any payment gateway integration. The project uses the Fake Store API to fetch and display real-time product data.

This project was created to strengthen fundamentals of JavaScript, DOM manipulation, API handling, and frontend logic.
